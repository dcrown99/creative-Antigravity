## 原因
データベース内に2種類の日付フォーマットが混在:
- `YYYY-MM-DD` (正しい): `2025-10-10`
- `YYYYMMDD` (誤り): `20251125`

文字列比較で `20251125` > `2025-12-01` となるため、11月の取引が12月のサマリーに含まれていた。

## 修正内容
1. 30件の取引日付を `YYYYMMDD` → `YYYY-MM-DD` に正規化
2. `transaction.service.ts` に日付フォーマット検証を追加（将来の予防）
