# Verification Ticket: Asset Edit Reflection Fix

## TICKET_20251221_1259

### 変更概要
- **問題**: 資産を編集・更新しても、資産一覧に変更が反映されない
- **原因**: Server Actionの`redirect()`で画面遷移後、`AssetsContext`のクライアント状態が更新されない

### 変更範囲

#### [MODIFY] `src/app/assets/edit/[id]/EditAssetClient.tsx`
- Server Action `updateAsset` から `updateAssetJson` に変更
- クライアントサイドの `handleSubmit` 関数を追加
- フォーム送信後に `router.push()` + `router.refresh()` を実行
- 送信中のローディング状態 (`isSubmitting`) を追加

#### [MODIFY] `src/contexts/AssetsContext.tsx`
- `useEffect` + `useRef` を追加し、`initialData` が変更された時にクライアント状態を同期
