# TICKET_20260109_1025

## 問題
資産一覧から「Direxion Daily Semiconductor Bull 3X Shares」(SOXL)を削除しようとすると、コンソールにエラーが表示され削除できない。

## 根本原因
`Dividend` テーブルの外部キー制約。SOXLには20件の配当レコードが紐づいており、`prisma.asset.delete()` が FK制約違反で失敗していた。

## 変更範囲
- `apps/money-master/src/services/asset.service.ts` - deleteAsset関数にカスケード削除ロジックを追加
- `apps/money-master/prisma/schema.prisma` - Dividend関連に `onDelete: Cascade` を追加

- [x] DB検証 (SOXL asset count: 0, SOXL dividends: 0)

## 修正内容
```typescript
// asset.service.ts - deleteAsset()
export async function deleteAsset(id: string) {
  // 1. Delete all dividends referencing this asset
  await prisma.dividend.deleteMany({ where: { assetId: id } });
  
  // 2. Nullify assetId in transactions (nullable field)
  await prisma.transaction.updateMany({
    where: { assetId: id },
    data: { assetId: null },
  });
  
  // 3. Delete the asset
  const result = await prisma.asset.delete({ where: { id } });
  // ...
}
```
