# 配当モニタリング Skill (Dividend Monitor)

MO (Altria Group) と BTI (British American Tobacco) の配当利回り・為替レート・購入タイミングを
多角的に分析してレポートするスキル。

## いつ使うか

- 「配当チェックして」「MO/BTI確認して」と言われた時
- 「買い増しタイミング教えて」と言われた時
- 高配当株の定期チェックを求められた時

## Step 1: DB照会 — 保有データ取得

// turbo
```
mcp_sqlite_execute_query("
  SELECT a.ticker, a.name, a.account, a.quantity, a.avgCost,
         a.currentPrice, a.currency, a.dividendRate, a.dividendYield,
         a.dividendFrequency, a.nextDividendDate
  FROM Asset a
  WHERE a.ticker IN ('MO', 'BTI')
    AND a.isArchived = 0
    AND a.account = 'NISA_GROWTH'
", read_only=true)
```

> [!NOTE]
> BTI特定口座 (3株) は対象外。NISA成長枠のみを分析対象とする。

// turbo
```
mcp_sqlite_execute_query("
  SELECT a.ticker, d.date, d.amount, d.currency
  FROM Dividend d
  JOIN Asset a ON d.assetId = a.id
  WHERE a.ticker IN ('MO', 'BTI')
    AND a.account = 'NISA_GROWTH'
  ORDER BY d.date DESC
  LIMIT 20
", read_only=true)
```

## Step 3: 税金計算ルール

| 銘柄 | 口座 | 米国源泉税 | 日本課税 | 税引後係数 |
|:---|:---|:---|:---|:---|
| **MO** | NISA成長 | 10% (米国法) | 0% (NISA非課税) | **×0.90** |
| **BTI** | NISA成長 | 0% (英国ADR) | 0% (NISA非課税) | **×1.00** |

- MO: 米国企業 → NISAでも米国10%源泉徴収は免除されない
- BTI: 英国ADR → 日英租税条約により米国源泉税0%、NISAで日本側も非課税

## Step 4: YOC計算

**Yield on Cost (取得価格利回り)** を以下の式で計算:

```
税引後YOC = (年間配当/株 × 税引後係数) ÷ 取得単価 × 100
年間手取配当 = 年間配当/株 × 税引後係数 × 保有株数
```

- **年間配当/株:** DB の `dividendRate` を基本値とし、Web検索で最新の増配発表があればそちらを優先
- **税引後係数:** Step 3 の税金計算テーブル参照 (MO: ×0.90, BTI: ×1.00)
- **取得単価:** DB の `avgCost`
- **保有株数:** DB の `quantity`

> [!TIP]
> 配当額はハードコードしないこと。DBとWeb検索の最新情報をその都度使ってリアルタイム計算する。

## Step 5: 5軸スコアリング (ファンダメンタルズ重視 = 合計100点)

> [!IMPORTANT]
> **配点設計思想:** 配当収入を主目的とした長期投資のため、ファンダメンタルズ系 (①③④) に75点を配分。
> テクニカルは「今の株価位置」の参考に留め (10点)、マクロは外部環境の確認程度 (15点) とする。

```
① 利回りシグナル   25点  ← 現在利回り・YOC・5年平均比較
② 株価位置         10点  ← 52週レンジ・200日MA (RSI削除)
③ バリュエーション  25点  ← Forward PER・目標価格・DCF
④ 配当安全性       25点  ← 連続増配・FCF配当性向・増配率
⑤ マクロ環境       15点  ← 金利・為替・セクター動向
```

### ① 利回りシグナル (25点)

```
# DBの配当履歴から過去の配当額推移を確認
# Web検索で「{ticker} 5 year dividend yield history」を取得
```

**現在利回り vs 5年平均利回り (/15):**
```
現在利回り ≥ 5年平均 + 1σ     → 15点 (歴史的に割安圏)
現在利回り ≥ 5年平均           → 12点 (平均的)
現在利回り ≥ 5年平均 × 0.9    → 8点  (やや割高)
現在利回り < 5年平均 × 0.9    → 4点  (割高)
```

**現在利回り vs YOC目標 (/10):**
```
市場利回り ≥ YOC目標 × 1.2   → 10点 (大幅に達成)
市場利回り ≥ YOC目標          → 7点  (目標クリア)
市場利回り ≥ YOC目標 × 0.9   → 4点  (惜しい)
市場利回り < YOC目標 × 0.9   → 0点  (未達)
```

**YOC目標:**
- MO: 税引後 ≥ 7% → 市場利回り ≥ 7.78%
- BTI: 税引後 ≥ 6% → 市場利回り ≥ 6.00%

> [!NOTE]
> **5年平均利回りの推定方法:**
> Web検索で直接取得できない場合は、以下で概算する:
> 1. Web検索で「{ticker} 5 year dividend yield history」を検索
> 2. 複数ソース (Macrotrends, GuruFocus, Koyfin) から平均利回り推移を確認
> 3. 明確な数値が得られない場合は「現在利回り vs YOC目標」のみでスコアリング (最大10点)

### ② 株価位置 (10点)

> **目的:** 株価が52週レンジと長期トレンドのどの辺にあるかを参照する。
> 純粋なテクニカル判定ではなく、ファンダメンタルズで「買い」と判断した場合の価格水準チェック。

**52週レンジ位置 (/5):**
```
位置% = (現在値 - 52週安値) ÷ (52週高値 - 52週安値) × 100

0-20%   → 5点 (年間底値圏)
20-40%  → 4点
40-60%  → 3点
60-80%  → 1点
80-100% → 0点 (年間高値圏)
```

**200日移動平均対比 (/5):**
```
株価 < 200MA × 0.95 (5%以上下)  → 5点
株価 < 200MA                    → 4点
株価 ≈ 200MA (±2%)              → 2点
株価 > 200MA × 1.05             → 1点
株価 > 200MA × 1.10             → 0点
```

### ③ バリュエーション (25点)

**Forward PER vs セクター平均 (/9):**
```
Forward PER < セクター平均 × 0.7  → 9点 (大幅割安)
Forward PER < セクター平均         → 7点
Forward PER ≈ セクター平均 (±10%)  → 4点
Forward PER > セクター平均 × 1.2   → 2点
Forward PER > セクター平均 × 1.5   → 0点
```

**アナリスト目標価格 (/8):**
```
現在値 < 目標 × 0.85  → 8点 (大幅アップサイド)
現在値 < 目標 × 0.95  → 6点
現在値 ≈ 目標 (±5%)   → 3点
現在値 > 目標          → 0点
```

**DCFフェアバリュー (/8):**
```
現在値 < DCF × 0.70  → 8点 (大きな安全余裕)
現在値 < DCF × 0.85  → 5点
現在値 < DCF          → 3点
現在値 > DCF          → 0点
```

### ④ 配当安全性 (25点)

**連続増配年数 (/9):**
```
≥50年 (Dividend King)      → 9点
≥25年 (Dividend Aristocrat) → 7点
≥10年                      → 4点
≥5年                       → 2点
<5年                       → 0点
```

> [!NOTE]
> **BTIの連続増配年数について:**
> BTIはGBPベース (LON: BATS) で **8年連続増配** (2018〜2026)。
> 参考: 215.60p(2021) → 217.80p(2022) → 230.89p(2023) → 235.52p(2024) → 240.24p(2025) → 245.04p(2026予)
> USD ADRベースでは為替変動により見かけ上の増減が生じるため、**GBPベースで判定すること。**

**配当性向 — FCFベース (/8):**
```
≤60%   → 8点 (余裕十分)
60-70% → 6点
70-80% → 4点
80-90% → 2点
≥90%   → 0点 (危険水域)
```

**直近増配率 (/8):**
```
≥5%  → 8点
3-5% → 6点
1-3% → 4点
0-1% → 1点
≤0%  → 0点 (減配/据え置き)
```

### ⑤ マクロ環境 (15点)

**米国10年金利トレンド (/5):**
```
低下傾向 (直近1ヶ月で-0.1%以上)  → 5点
横ばい (±0.05%)                  → 3点
上昇傾向 (直近1ヶ月で+0.1%以上)  → 0点
```

**USD/JPY 為替 (/5):**
```
円高方向 (¥140以下 or 低下傾向)  → 5点
安定圏 (¥140-155)               → 3点
急激な円安 (¥155以上 + 上昇傾向) → 1点
```

**セクターローテーション (/5):**
```
ディフェンシブ流入 (Consumer Staples好調)  → 5点
中立 (明確なトレンドなし)                  → 3点
グロース集中 (テック・AI主導)              → 1点
```

## Step 6: 総合判定

### スコア集計

```
合計 = ① + ② + ③ + ④ + ⑤ (最大100点)
```

| 合計スコア | 判定 | アクション |
|:---|:---|:---|
| **80〜100** | 🟢 積極買い | 即座に買い増し検討。具体的な株数を提案 |
| **60〜79** | 🟡 監視継続 | 良い水準だが急がない。2週間後に再チェック |
| **40〜59** | 🟠 待ち | 割安感がない。目標買い価格を提示して待機 |
| **0〜39** | 🔴 見送り | 割高または環境悪化。手を出さない |

### 目標買い価格の算出

各銘柄について、スコアが60点以下の場合は以下を提示:

```
目標買い価格 = 年間配当/株 ÷ 目標利回り

MO: 年間配当 ÷ 7.78% (税引後YOC 7%の水準)
BTI: 年間配当 ÷ 6.00% (税引後YOC 6%の水準)
```

> [!NOTE]
> 目標買い価格は最新の年間配当/株 (Step 4 で確定した値) を使ってリアルタイム算出すること。

### 買いタイミング判定 (ファンダメンタルズベース)

既存の5軸スコアから「今が買いどきか」を判定する。新たなデータソースは不要。

**判定ロジック:**

| 条件 | シグナル | 解説 |
|:---|:---|:---|
| 合計 ≥ 80点 | 🟢 即エントリー | ファンダ全面好条件。待つ理由なし |
| 合計 60-79点 かつ ③バリュエーション ≥ 18/25 | 🟡 打診買い | スコア中位だがバリュエーション面で大幅割安 |
| 合計 60-79点 かつ ①利回り ≥ 20/25 | 🟡 打診買い | スコア中位だが利回りが歴史的高水準 |
| 合計 40-59点 | 🟠 目標価格で指値待機 | 目標買い価格を提示 |
| 合計 ≤ 39点 | 🔴 見送り | エントリー不可 |

**権利落ち日コンテキスト (補足情報):**

DB の `nextDividendDate` から次回権利落ち日までの残日数を計算し、レポートに表示する。

```
権利落ち日まで14日以内の場合:
  ⚠️「権利落ち後の株価調整を待つ選択肢あり」と注記

権利落ち日から14日以内（直後）の場合:
  💡「配当落ち後のため、YOCベースで有利な水準の可能性あり」と注記
```

> [!TIP]
> 権利落ち日の注記は「推奨/非推奨」ではなく「判断材料の提示」に留める。
> MO/BTIのような大型成熟株では、権利日前後の株価変動は小型株ほど顕著ではない。

## 注意事項

> [!CAUTION]
> **投資判断はユーザーの責任で行われます。**
> このスキルは情報整理と分析支援を目的としており、投資助言ではありません。

> [!WARNING]
> **BTI配当のGBP/USD為替リスク:**
> BTIの配当はGBPで宣言され、ADR経由でUSD換算されます。
