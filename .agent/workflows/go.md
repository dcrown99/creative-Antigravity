# /go - タスク再開ワークフロー

1. `@AI_STATUS.md` を読み込み、Current Focus / Blockers / Last Ticket を確認する
2. Last Ticket のリンク先チケットを確認し、前回の完了状況を把握する
3. Blockers があれば、その解決を最優先で対応する
4. Current Focus に基づき、作業を再開する

> 補足: 必要に応じて `ARCHITECTURE.md` や `PROJECT_MAP.md` も参照すること
